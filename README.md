## 💡 What is Trevia?

**Trevia** is a modern AI-powered travel assistant designed to simplify how people explore India. Instead of generic plans, it generates **personalized travel experiences** based on user preferences like mood, budget, and interests.

Whether you're looking for adventure, culture, or peace — Trevia adapts to you.

---

## 🌟 Core Highlights

- ✨ AI-generated **custom itineraries**
- 📍 Curated **destination insights**
- 💸 Smart **budget tracking system**
- 🔎 Discover **hidden local experiences**
- 🛡️ Travel with **region-specific safety guidance**
- 📬 Built-in **user support/contact system**

---

## 🛠️ Setup Instructions

### Requirements
- Node.js (v18+)
- npm

### Run Locally

```bash
git clone https://github.com/sathwik27-ai/Trevia-project.git
cd Trevia-project
npm install
npm run dev
````

Open 👉 http://localhost:3000

---

## 🚀 Production Build

```bash
npm run build
npm start
```

---

## ☁️ Deployment

Hosted on **Vercel** with automatic deployment on every push to `main`.

Manual deploy:

```bash
npx vercel --prod
```
